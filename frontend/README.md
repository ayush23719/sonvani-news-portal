# Frontend

React + Vite + TypeScript frontend for the Hindi News Portal.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run format
```

## Current Scope

Foundation only:

- App providers
- React Router
- Material UI theme
- Noto Sans Devanagari font
- App shell
- Header, navigation, and footer placeholders

Homepage sections and article pages are intentionally not implemented yet.
