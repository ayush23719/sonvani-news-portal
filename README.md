# Hindi News Portal

Production-oriented Hindi news portal foundation using React, Vite, TypeScript, Material UI, TanStack Query, React Router, AWS SAM, Lambda, API Gateway, DynamoDB, S3, and CloudFront.

## Current Milestone

Project Foundation only.

- Frontend scaffold is ready.
- Backend SAM scaffold is ready.
- Shared folders, linting, formatting, environment examples, and docs are ready.
- Homepage sections, article pages, and Lambda business logic are intentionally not implemented yet.

## Quick Start

```bash
npm install --prefix frontend
npm install --prefix backend
npm run build --prefix frontend
npm run build --prefix backend
```

## Local Frontend

```bash
npm run dev --prefix frontend
```
